In-Depth: Smart Containers

Project Type:

Web Technologies for Mobile using the Dojo Toolkit

Year Completed:

2012

About:

In-Depth: Smart Containers are a response to all the rotten food, unneccessary purchases, and special cravings you've ever had. This Internet of Things web app prototype simulates the ability to see what's in your kitchen from wherever you are, just by checking your smart phone. Prototyped with the Dojo Toolkit, an Arduino, Processing, an Ikea jar, and some flour, this was only functional so long as I was willing to leave my beloved computer in the kitchen.
