The Fibonacci Sequence

Project Type:

iOS for iPad using the C4 Framework

Year Completed:

2013

About:

This experiment visualizes the relationship between the Fibonacci numbers and the Fibonacci spirals in a pine cone. The seeds of a pine cone form two sets of Fibonacci spirals in opposite directions that originate in the base of a pine cone. The number of spirals in each direction count up to a Fibonacci number: 8 and 13. Visit the source code on my GitHub.
